# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | Yes       |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities by emailing **security@glueckkanja.com**. Include:

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Any known mitigations

You will receive an acknowledgement within 3 business days and a resolution plan within 10 business days.

## Scope

This policy covers the `terraform-provider-gkvm` binary and its Go source code. Third-party dependencies are out of scope — report those upstream.
